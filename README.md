# Task Three - Daily Task Scheduler

Yarn Workspaces를 사용한 모노레포 프로젝트

## 프로젝트 구조

```
task-three/
├── apps/
│   ├── frontend/    # Next.js 앱
│   └── backend/     # NestJS 앱
├── packages/
│   └── shared/      # 공통 타입 및 유틸리티
└── package.json     # 루트 워크스페이스 설정
```

## 시작하기

### 설치

```bash
yarn install
```

### 개발 서버 실행

```bash
# Frontend만 실행 (포트 3000)
yarn dev

# Backend만 실행 (포트 3000)
yarn dev:backend

# Frontend와 Backend 동시 실행
yarn dev:all
```

### 빌드

```bash
# Frontend 빌드
yarn build

# Backend 빌드
yarn build:backend

# 모든 앱 빌드
yarn build:all
```

### 테스트

```bash
# 모든 워크스페이스 테스트 실행
yarn test
```

## 워크스페이스

- **@task-three/frontend**: Next.js 기반 프론트엔드 앱
- **@task-three/backend**: NestJS 기반 백엔드 API
- **@task-three/shared**: Frontend와 Backend에서 공유하는 타입 및 유틸리티

# 기능 명세
메인 페이지
메인 페이지에는 다음 세 영역이 존재한다.
날짜(일) 선택 영역
할 일 목록 영역 : 사용자가 태스크를 계속 쌓아두는 영역
스케줄 영역 : 오늘 실행할 태스크를 최대 3개까지 지정하는 영역

할 일 추가
할 일 추가 버튼을 클릭하면 모달이 나타난다.
할 일 추가에는 총 3가지 정보가 필요하다.
할 일 이름(Required) - 20자 제한
시작 시간(Required)
끝낼 시간(Required)
사용자가 추가 버튼을 누르면 태스크가 할 일 목록에 누적된다.
오늘 완료하지 못한 일은 다음 날로 넘어간다.
스케줄 추가
스케줄에는 최대 3개의 할 일이 올 수 있다.
할 일 목록의 할 일을 드래그앤 드롭으로 스케줄 영역에 놓을 수 있다.
드래그 앤 드롭시 할 일 이름, 시작 시간, 끝낼 시간 정보가 있는 모달이 나타난다.
모달에서 확인을 하면 스케줄에 등록된다.

할 일, 스케줄 수정
할 일, 스케줄 을 클릭하여 나타난 바텀 시트에서 할 일을 수정할 수 있다.
할 일 이름, 시작 시간, 끝낼 시간이 수정 가능하다.

할 일, 스케줄 삭제
할 일, 스케줄 을 클릭하여 나타난 바텀 시트에서 할 일을 삭제할 수 있다.
스케줄을 드래그 앤 드롭으로 할 일 목록으로 이동 시킬 수 있다.

날짜 선택
날짜를 선택하여 다른 날의 할 일 목록, 스케줄을 볼 수 있다.
어떤 날짜든 할 일 목록, 스케줄은 수정 가능하다

설정 페이지
설정 페이지에서 시간 설정이 가능하고, 설정을 해야 메인 페이지로 진입할 수 있다
시간 설정
스케줄의 시작 시간, 마무리 시간을 지정할 수 있다.

특이사항
최초 시간 설정을 안하면 홈화면은 안넘어간다
일단 어떤 날짜든 스케줄, 할일을 수정 가능하도록 한다
스케줄의 시작, 끝 시간이 겹치는 것을 허용한다
